# Drift Boss - Java Swing Version

A 2D drifting game built with Java Swing, inspired by the popular web game Drift Boss.

## Features

- **Auto-Forward Movement**: Car moves forward automatically
- **Drift Mechanics**: Smooth turning with drift physics
- **Procedural Platforms**: Randomly generated platforms with turns
- **Camera Tracking**: Camera follows the car as it progresses
- **Score System**: Track your distance traveled
- **Multiple Controls**: Mouse or keyboard controls

## Controls

### Mouse
- **Click Left Side**: Turn left
- **Click Right Side**: Turn right
- **Release**: Stop turning

### Keyboard
- **Left Arrow / A**: Turn left
- **Right Arrow / D**: Turn right
- **Space**: Restart (when game over)

## How to Run

1. Compile the game:
   ```bash
   javac HillClimbRacing.java
   ```

2. Run the game:
   ```bash
   java HillClimbRacing
   ```

## Game Mechanics

- The car moves forward automatically
- Click or use arrow keys to turn left or right
- Stay on the platforms to survive
- Platforms have random turns and colors
- If you fall off a platform, the game ends
- Score is based on distance traveled
- New platforms are generated as you progress

## Tips

- Time your turns carefully to stay on platforms
- The car has drift physics, so plan your turns in advance
- Platforms get more challenging as you progress
- Try to maintain momentum for smoother turns

## Requirements

- Java 8 or higher
- No external dependencies (uses only Java Swing)
