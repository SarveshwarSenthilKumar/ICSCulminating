# Part 1 - SDLC Project Write Up

## Game Name: Cannonball Run: Highway Chase

### Description

This game is a high-speed, lane-based racing game built in Java Swing where the player controls a car on a multi-lane highway which spans infinitely while avoiding opponent vehicles on the highway as well as collecting power-ups. The user has an interactive display where they can see their score, speed, as well as the number of lives. The game is built with an in-depth collision detection system to track opponent vehicle collisions as well as with powerup pickups. The objective of the game is to gain as high a score as possible by avoiding collisions and speeding up to increase score faster.

The main theme of the project will be surrounding cars and collision detection to achieve a project with simple mechanics yet one that is fun and addicting to play. I chose this project as I am a car enthusiast and this felt like my forte as well as it allows me to tone back the excessive cyberpunk-esque themes of my last project and allow me to focus on more menial details. This project will also allow me to implement the various systems and concepts learned in class in an effective and sophisticated manner.
